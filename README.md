
# Expense Tracker App 

This app can be used to track your expenses,
store record of your all spendings.


## Features

- Undo option 
- Adding option
- Swipe deletion
- Multi Screen
- Cross platform


## Lessons Learn

Lessons I learned by creating this app :-

- Appbars 
- Snackbars
- Dialog boxes
- ModalBottom Sheet
- Calender to pic date
- DropDown Box
- Widgets (Statefull & Stateless)
- if else Statements
- Types of buttons 
- Styling of Widgets
- Mapping of data
