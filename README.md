
# Expense Tracker App 

This app can be used to track your expenses,
store record of all your spendings.


## Features

- Undo option 
- Adding option
- Swipe deletion
- Multi Screen
- Cross platform


## Lessons Learn

Lessons I learned by creating this app :-

- Appbars 
- Snackbars
- Dialog boxes
- ModalBottom Sheet
- Theme of app
- Calender to pic date
- DropDown Box
- Widgets (Statefull & Stateless)
- if else Statements
- Types of buttons 
- Styling of Widgets
- Mapping of data
